# プロジェクト概要

このプロジェクトは、Express.jsを使用したTypeScriptのWebサーバーのプレイグラウンドです。

## 技術スタック

- **言語:** TypeScript
- **フレームワーク:** Express.js
- **パッケージマネージャー:** npm

## プロジェクト構造

- `package.json`: プロジェクトの依存関係とスクリプトを定義します。
- `src/server.ts`: Expressサーバーをセットアップし、基本的なルートを定義するエントリーポイントです。
- `node_modules/`: プロジェクトの依存関係がインストールされるディレクトリです。

## サーバーの動作

- `src/server.ts`で定義されたサーバーは、ポート3000でリッスンします。
- ルートURL (`/`) へのGETリクエストに対して、`hello world`という文字列を返します。
